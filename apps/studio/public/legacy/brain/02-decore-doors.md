# Decore.com — Doors & Drawer Fronts

Every door and drawer-front style Decore sells that is relevant to Saunders Wood Works. Grouped by construction method. Model codes in parentheses are the exact codes on Decore's standard order form.

> **Glass variants.** A suffix of `G` (wood) or `X` (miter-joint) indicates a glass-door version with rubber glass stops. The base style is otherwise identical. Example: `520X` = glass version of `520` (Newbury).
> **French Lite variants.** Suffix `F` on a wood style (e.g., `841F`) indicates a French-lite variant. The order form asks how many lites.

## 5. Material / grade options (for WOOD styles)

When a WOOD style is ordered, you must pick a **Face** material and a **Frame** material. Possible options on the Decore order form:

- Maple Select
- Maple Paint Grade (MDF Panel / Hardwood Frame)
- White Oak Select
- Rift White Oak
- Walnut Select
- Alder Select
- Cherry Select
- Red Oak Select
- Hardwood Paint Grade (MDF Panel / Hardwood Frame)
- Aspen
- Rift White Oak Veneer

Panel and frame can differ — e.g., a paint-grade panel with a hardwood-frame build is the standard way to order a painted shaker.

## 6. Spec fields that apply to every WOOD order

These are on the order form and must be captured in any intake that feeds Decore:

- **Panel Detail** — Standard (default) or Other (specify)
- **Inside Edge Detail** — Standard (default) or Other (specify)
- **Outside Face Edge Detail** — Standard (default) · Square Face Edge · Other
- **Outside Back Edge Detail** — Square Back Edge (default) · Other
- **Bore Pattern** — Bore A (5mm tab) · Bore B (3mm tab) · Other · None
- **Bore Position** — Left · Center · Right · None
- **DF Grain Direction** — Standard (default) · Vertical *(Clear and Stain finishes only)*
- **Size** — products manufactured to the **actual** sizes provided (no allowance math by Decore)

See `05-decore-order-form.md` for the full field map.

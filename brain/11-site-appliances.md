# Site Appliances Reference

_Last updated: 2026-04-19_

Combined appliance specification material (models, clearances, ventilation, electrical rough-in) used during design and field verification for kitchens and adjacent built-ins.

## Source PDF

- **`docs/combined-appliances.pdf`** — full “COMBINED - Appliances” reference (also listed in Agent Brain → Source Documents).

Treat this PDF as the **shop-side companion** to manufacturer cut sheets. For authoritative cut-outs on a live job, prefer the **specific model’s** installation PDF uploaded via intake **Appliance Packets**.

## Intake form impact (`intake.html`)

| Intake area | How appliances matter |
|-------------|----------------------|
| **Appliance Packets** | Every refrigerator, range, wall oven, microwave, cooktop, hood, dishwasher, and warming drawer should have a packet when dimensions drive casework. The combined PDF helps agents ask for the *right* submittal if something is missing. |
| **Dimensions** | Rough opening width/height/depth, ventilation chase, toe-kick vs. adjustable feet, panel-ready vs. integrated overlay, and adjacent clearances (hinge swing, handle projection). |
| **Site readiness** | Delivery path (stairs, elevator), floor loading for sliders, temporary power, and protection of finished floors during appliance set. |

## Agent playbook hooks

- When a user asks “will this Sub-Zero fit this opening?”, require **model number + installation PDF** or intake dimensions — never guess from marketing renderings alone.
- Cross-reference **hood CFM**, **make-up air**, and **combustion air** requirements with local code; the combined PDF is a starting point, not a code substitute.
- If appliance PDFs conflict with the combined reference, **the manufacturer PDF uploaded on the intake wins**.
